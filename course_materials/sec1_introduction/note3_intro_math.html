3.2 微分¶

$y=f(x)$$f'(x)$

  • $(x^n)'=nx^{n-1}$
  • $(\sin x)'=\cos x$, $(\cos x)'=-\sin x$
  • $(\exp x)'=\exp x$
  • $(\log x)'=\frac{1}{x}$

  • $x=x_0$
  • $x_{n+1}=x_{n}-\eta f'(x_{n})$, $\eta$

def gradient_descent(start,eta,iter):    # start / eta / iter
    x = start
    for i in range(iter):
        x = x - eta*(2*x)
