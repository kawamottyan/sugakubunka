2.4　Newtonブースティング法のlightgbmによるデモ¶

A. Newtonブースティング決定木回帰モデル¶

Newtonlightgbm packagealgorithm:

Input :

  • $\eta>0$ :
  • $\nabla_h L$ : $L$1
  • $\nabla^2_h L$ : $L$2
  • $M$ :

Process :

  1. $\hat{h}(x)\leftarrow \bar{y}$
  2. for $t$ in $1,\cdots,M$:
  3. $h_{t}$$\nabla_h L$, $\nabla^2_h L$
  4. $\hat{h}(x)\leftarrow\hat{h}(x)+\eta h_{t}(x)$
  5. return $\hat{h}(x)$
