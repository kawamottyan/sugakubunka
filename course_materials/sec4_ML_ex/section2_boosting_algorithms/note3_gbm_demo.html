2.3　勾配ブースティング法のsklearnによるデモ¶

A. 勾配ブースティング決定木回帰モデル¶

sklearn packagealgorithm:

Input :

  • $\eta>0$ :
  • $\nabla_h L$ : $L$
  • $M$ :

Process :

  1. $\hat{h}(x)\leftarrow \bar{y}$
  2. for $t$ in $1,\cdots,M$:
  3. $\mathcal{D}':=\{(x_1,2(y_i-\hat{h}(x_i))),\cdots,(x_n,2(y_n-\hat{h}(x_n)))\}$$h_{t}(x)$
  4. $w_{t}:=\mathrm{argmin}_{w\in\mathbb{R}}L(\hat{h}+wh_{t})$
  5. $\hat{h}(x)\leftarrow\hat{h}(x)+\eta w_{t}h_{t}(x)$
  6. return $\hat{h}(x)$
